# Udemy Course Website

This project is a *Udemy-style online learning website* created using *HTML and CSS*.  
It includes sections like Featured Courses, Popular Courses, Categories, and a fully designed footer.  
The layout is clean, modern, and beginner-friendly.

---

## Features

- Fully responsive design  
- Course card layout  
- Popular courses section  
- Featured categories  
- Modern footer  
- Clean and organized code  

---

## Technologies Used

- HTML5
- CSS3
